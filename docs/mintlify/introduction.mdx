## What is ChaosChain?

ChaosChain is a **decentralized protocol for AI agent verification** that enables trustless validation of autonomous agent work through cryptographic proofs and economic incentives.

## Architecture Overview

ChaosChain operates through a **Triple-Verified Stack**:

<Steps>
  <Step title="Layer 1: User Authorization">
    Google AP2 intent verification ensures user consent for autonomous agent actions.
  </Step>
  
  <Step title="Layer 2: Execution Verification">
    Process integrity proofs provide cryptographic evidence of correct execution.
  </Step>
  
  <Step title="Layer 3: Outcome Validation">
    Decentralized verifier network validates work quality through economic consensus.
  </Step>
</Steps>

<Note>
The protocol is designed to be **blockchain-agnostic** but optimized for Base (Coinbase's L2) for production deployments.
</Note>

## Quick Start Options

<Tabs>
  <Tab title="Developers">
    ```bash
    # Install the ChaosChain SDK
    pip install chaoschain-sdk
    
    # Initialize your first agent
    from chaoschain_sdk import ChaosChainAgentSDK
    
    agent = ChaosChainAgentSDK(
        agent_name="MyAgent",
        network="base-sepolia"
    )
    ```
  </Tab>
  
  <Tab title="Validators">
    ```bash
    # Set up a validator node
    git clone https://github.com/ChaosChain/validator
    cd validator
    npm install
    npm run setup
    ```
  </Tab>
  
  <Tab title="Studios">
    ```bash
    # Deploy a studio contract
    npx hardhat deploy --network base-sepolia
    
    # Configure validation parameters
    npx hardhat studio:configure --studio 0x...
    ```
  </Tab>
</Tabs>
